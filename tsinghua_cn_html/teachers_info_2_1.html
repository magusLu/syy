关志成教授 博士生导师

电话：0755-2603-6385　　邮箱：guanzc@sz.tsinghua.edu.cn 办公地址：清华大学深圳研究生院D303

中国电机工程学会副理事长，高电压技术专委会副主任委员，中国电工技术学会副理事长，电工产品环境技术专委会副主任委员，联合国教科文继续工程教育主任教席，清华学报（自然科学版）副主任编委，清华大学学报（英文版）副主任编委。“高电压技术”名誉编委。

个人简历
1970年清华大学电机系本科毕业后留校工作 ;
1978年考取本校研究生脱产学习 ;
1981年、1984年分获清华大学工学硕士和工学博士学位 ;
1987~1989在英国曼彻斯特理工学院作访问学者 ;
1991年评聘教授，1994年评聘博士生导师 ;
1999年7月-10月在澳大利亚墨尔本大学作访问教授。
教学
主要课程：
  • 文化创意产业概论
  • 低碳城市专题讲座
研究领域
主要从事高电压与绝缘技术研究，取得多项科技成果，已获国家奖2项，省部级奖10项。主要奖项有：
  • “沿染污介质表面放电研究”获原国家教委科技进步奖（甲类）一等奖，国家自然科学奖三等奖；
  • “高压合成绝缘子”获原国家教委科技进步奖二等奖，国家科技进步奖二等奖；
  • “合成棒型悬式绝缘子”获国家重大技术装备成果奖；
  • “RTV防污闪憎水长效涂料”获国家首届电工新产品开发进步奖，并被列为国家重点科技成果推广项目；
  • “防污闪涂料应用技术”获天津市科技进步二等奖；
  • “供电设备状态维修与实践”获原电力部科技进步二等奖；
  • “电力设备防污闪技术”获原电力部科技进步三等奖；
研究成果
1.《人文三十年》香港社会科学出版社，2011年；
2.《深圳文化蓝皮书》（副主编）中国社会科学出版社，（2003～2010年）；
3.《深圳经济蓝皮书》（副主编）社会科学文献出版社，（2003～2010年）；
4.《深圳社会蓝皮书》（副主编）社会科学文献出版社，（2003～2010年）；
5.《建设国家自主创新型城市的战略选择》（《深圳经济蓝皮书》，社会科学文献出版社，2006年）
6.《文化与科技结合主导下的深圳创意城市建设》（《深圳文化蓝皮书》，中国社会科学出版社，2010年）
7.《从经济特区到绿色低碳特区》（《深圳低碳发展报告》，海天出版社，2013年）；
8.《深圳低碳发展报告》（主编）海天出版社，2013年；
主要荣誉
1991年被授予“做出突出贡献的中国博士学位获得者”称号 ;
1996年被评为“国家有突出贡献中青年专家”，获“优秀留学回国人员”表彰。
